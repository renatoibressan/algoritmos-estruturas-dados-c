# Algoritmos e Estruturas de Dados em C

Bem-vindo ao repositório de estudos de **Algoritmos e Estruturas de Dados** em C! 🌟

Este projeto reúne exemplos didáticos em C para explorar conceitos fundamentais de programação, lógica, ordenação, busca e estruturas de dados.

## 📚 O que você encontra aqui

- **Algoritmos de busca**
  - Busca Linear
  - Busca Binária
  - Busca Ternária
  - Busca Saddleback

- **Algoritmos de ordenação**
  - Bubble Sort
  - Selection Sort
  - Insertion Sort
  - Merge Sort
  - Quick Sort
  - Heap Sort
  - Radix Sort
  - Bucket Sort
  - Shell Sort

- **Recursão e problemas clássicos**
  - Fatorial
  - Fibonacci
  - Torre de Hanoi
  - MDC
  - Potenciação

- **Estruturas de dados**
  - Lista estática e dinâmica
  - Pilha estática e dinâmica
  - Fila estática e dinâmica
  - Árvore binária
  - Tabela hash

## 🚀 Como usar

Cada pasta contém código `C` e um `main` de exemplo para testar o funcionamento.

1. Navegue até a pasta desejada.
2. Compile com `gcc` ou sua ferramenta preferida.
3. Execute o binário gerado.

Exemplo:

```bash
cd "Algoritmos/Busca/Busca Binária"
gcc main.c busca_binaria.c -o busca_binaria
./busca_binaria
```

## 🧠 Aprendizados

- Entender como os algoritmos funcionam passo a passo.
- Comparar eficiência entre diferentes métodos de ordenação e busca.
- Ver na prática a diferença entre estruturas de dados estáticas e dinâmicas.
- Aprender a implementar recursão e resolver problemas clássicos.
- Solidificar fundamentos de `C`, como ponteiros, alocação de memória e manipulação de listas.
